## üêõ Troubleshooting

### Common Issues:

**"Gemini API not working"**
- Check API key is correct
- Verify you have free tier quota
- Check network connection

**"Notifications not showing"**
- Request permission in Settings
- Check browser supports notifications
- Verify Service Worker is registered

**"Audio not playing"**
- Check browser autoplay policy
- User must interact first
- Verify audio URLs are correct

**"Translations not working"**
- Check language code is correct
- Verify translation file exists
- Check t() function is called correctly
